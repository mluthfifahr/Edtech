# 🎒 Fun Learning EdTech Web App

A modern, gamified, and highly interactive educational platform designed specifically for elementary school (SD) students. This platform uses a **decoupled (headless) architecture** to separate the powerful administrative backend from the playful, animation-rich frontend.

## 🚀 Features

### 🧑‍🎓 Student Interface (Frontend)
- **Gamified Dashboard:** Interactive and child-friendly map/grid layout to choose school subjects.
- **Micro-interactions & Animations:** Engaging user experience using smooth spring animations, hover effects, and shake feedbacks powered by Framer Motion.
- **Diverse Learning Methods:** Supports traditional multiple-choice quizzes and visual "Tebak Gambar" (image guessing) games.
- **Victory Feedback:** Beautiful success screens with interactive confetti celebration components.

### 💼 Back Office Admin Panel (Backend)
- **Secure Authentication:** Protected administrative routes powered by Laravel Sanctum.
- **Subject Management:** Full CRUD system to add, edit, or remove school subjects.
- **Dynamic Question Bank:** Media-ready question management that supports rich-text assets and image uploads for visual games.

---

## 🛠️ Tech Stack

**Frontend Architecture:**
- React (Vite) + TypeScript
- Tailwind CSS (Playful styling)
- Framer Motion (Fluid animations)
- Zustand (Lightweight global state management)
- Axios (API Client with Interceptors)

**Backend Architecture:**
- Laravel (RESTful API Engine)
- Laravel Sanctum (Token-based Authentication)
- MySQL / PostgreSQL (Relational Database)
